# 👑 IGRIS — The King’s AI System

> *“Born from ambition. Forged through code. Designed to obey only one commander.”*

IGRIS is a next-generation AI-powered tactical assistant built for automation, intelligence workflows, file analysis, and real-time conversational interaction. It combines a modern chat interface with multimodal input, session memory, and extensible AI tooling.

## ⚡ Live Demo
🌐 https://igris-core.vercel.app  


---

## 🧠 Overview

IGRIS is not just a chatbot — it is a **command system interface** designed to:

- Automate tasks through AI commands
- Analyze files (images, PDFs, code, documents)
- Maintain session-based memory
- Provide real-time conversational intelligence
- Support multimodal inputs (text + files)
- Run as a Progressive Web App (PWA)

---

## 🚀 Core Features

### 💬 AI Command Interface
- ChatGPT-style conversation system
- Session-based chat history
- Markdown + code rendering support
- Streaming-friendly UI structure

### 📎 Multimodal Intelligence
- Image analysis
- PDF & document parsing
- Code file interpretation
- Smart file type detection

### 🧩 Session System
- Create / rename / delete sessions
- Pin important chats
- Persistent chat history
- Tactical session control panel

### 📱 PWA Support
- Installable on mobile & desktop
- Offline-capable service worker
- App-like fullscreen experience
- Custom manifest configuration

### 🎨 UI System
- Dark tactical cyber UI
- Theme accent system (violet / crimson / silver)
- Animated micro-interactions
- Glassmorphism chat bubbles
- Mobile-optimized rendering

### 📤 Export System
- Export chat as Markdown
- Structured conversation logs
- Timestamped session archives

## 🧱 Tech Stack

- **Frontend:** React + TypeScript
- **Styling:** TailwindCSS
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Build Tool:** Vite
- **PWA:** Service Worker + Web Manifest
- **File Analysis:** Custom API layer
